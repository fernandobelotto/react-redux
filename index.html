React Redux

Ligações oficiais do React para Redux

Comece agora

Oficial

React Redux é mantido pela equipe Redux, e é atualizado com as APIs mais recentes do Redux e React.

Previsível

Projetado para trabalhar com o modelo de componente do React. Você define como extrair os valores de que seu componente precisa do Redux, e seu componente os recebe como props.

Encapsulado

Cria componentes de wrapper que gerenciam a lógica de interação com a store para você, para que você não precise lidar.

Otimizado

Implementa automaticamente otimizações de desempenho complexas, para que seus componentes sejam renderizados novamente somente quando os dados necessários forem realmente alterados.
